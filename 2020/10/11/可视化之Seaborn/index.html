Step 3 调色板#

就是对图片的颜色进行配置。选择指定的颜色风格。通过color_palette()对颜色板进行处理，返回颜色，并对图片进行上色。

调色板#

主要使用以下几个函数设置颜色：
color_palette() 能传入任何Matplotlib所有支持的颜色
color_palette() 不写参数则默认颜色
set_palette() 设置所有图的颜色

主要使用以下几个函数设置颜色：

color_palette() 基于RGB设置颜色的接口#

能传入任何Matplotlib所有支持的颜色。基于RGB原理设置颜色的接口，可接收一个调色板对象作为参数，同时可以设置颜色数量

Returns:
palette : list of RGB tuples. 返回一个颜色三元组列表（默认是6个）

Step5 按定制需求图#

Seaborn 一共拥有 50 多个 API 类，相比于 Matplotlib 数千个的规模，可以算作是短小精悍了。其中，根据图形的适应场景，Seaborn 的绘图方法大致分类 6 类，分别是：关联图、类别图、分布图、回归图、矩阵图和组合图。而这 6 大类下面又包含不同数量的绘图函数。

关联图：API类#
关联性分析 介绍
relplot 绘制关系图
scatterplot 多维度分析散点图
lineplot 多维度分析线形图
类别图#
  • 分类散点图：stripplot() (kind="strip")swarmplot() (kind="swarm")
  • 分类分布图：boxplot() (kind="box")violinplot() (kind="violin")boxenplot() (kind="boxen")
  • 分类估计图：pointplot() (kind="point")barplot() (kind="bar")countplot() (kind="count")

seaborn作为一个库，里面包含了很多类，如PairGrid类。创造相应类的实例（传参)，就可以调用类里面的函数，参数，进行个性化设置。

  1. 导入库 2. 创造类实例 3. 调用实例的参数和方法 4. show()

分布图#

分布图主要是用于可视化变量的分布情况，一般分为单变量分布和多变量分布。当然这里的多变量多指二元变量，更多的变量无法绘制出直观的可视化图形。

单变量分布

distplot，kdeplot

多变量分布图

jointplot，pairplot
